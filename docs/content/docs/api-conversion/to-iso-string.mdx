Convert to ISO 8601 string format.

## Syntax

```typescript
toISOString(): string
```

## Returns

Returns an ISO 8601 formatted string in UTC timezone.

## Examples

### Basic Usage

```typescript
import { fdu } from "@pyyupsk/fdu";

const date = fdu("2025-09-30T14:30:45");
date.toISOString(); // '2025-09-30T14:30:45.000Z'
```

### Different Inputs

```typescript
// From date string
fdu("2025-09-30").toISOString();
// '2025-09-30T00:00:00.000Z'

// From timestamp
fdu(1727704245000).toISOString();
// '2025-09-30T14:30:45.000Z'

// Current time
fdu().toISOString();
// '2025-10-04T12:00:00.000Z' (example)
```

### Storing in Database

```typescript
const date = fdu("2025-09-30T14:30:45");

const record = {
  id: 1,
  createdAt: date.toISOString(),
};

// Save to database
await db.save(record);
```

### API Requests

```typescript
const date = fdu("2025-09-30");

const response = await fetch("/api/events", {
  method: "POST",
  body: JSON.stringify({
    eventDate: date.toISOString(),
  }),
});
```
