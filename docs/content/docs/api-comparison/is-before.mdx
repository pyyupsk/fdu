---
title: isBefore()
description: Check if the date is before another date
---

Checks if the current date-time instance is before another date instance by comparing timestamps.

## Syntax

```typescript
isBefore(other: FduInstance): boolean
```

## Parameters

- **other**: `FduInstance` - Date instance to compare with

## Returns

Returns `true` if the date is before the given date, `false` otherwise.

## Examples

### Basic Comparison

```typescript
import { fdu } from "@pyyupsk/fdu";

const date1 = fdu("2025-09-30");
const date2 = fdu("2025-10-01");

date1.isBefore(date2); // true
date2.isBefore(date1); // false
```

### Time Comparison

```typescript
const date1 = fdu("2025-09-30T10:00:00");
const date2 = fdu("2025-09-30T14:00:00");

date1.isBefore(date2); // true (same day, but earlier time)
```

### Practical Example

```typescript
const now = fdu();
const deadline = fdu("2025-12-31");

if (now.isBefore(deadline)) {
  console.log("Still time before deadline!");
}
```
