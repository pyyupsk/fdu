---
title: isAfter()
description: Check if the date is after another date
---

Checks if the current date-time instance is after another date instance by comparing timestamps.

## Syntax

```typescript
isAfter(other: FduInstance): boolean
```

## Parameters

- **other**: `FduInstance` - Date instance to compare with

## Returns

Returns `true` if the date is after the given date, `false` otherwise.

## Examples

### Basic Comparison

```typescript
import { fdu } from "@pyyupsk/fdu";

const date1 = fdu("2025-10-01");
const date2 = fdu("2025-09-30");

date1.isAfter(date2); // true
date2.isAfter(date1); // false
```

### Time Comparison

```typescript
const date1 = fdu("2025-09-30T14:00:00");
const date2 = fdu("2025-09-30T10:00:00");

date1.isAfter(date2); // true (same day, but later time)
```

### Practical Example

```typescript
const event = fdu("2025-09-30");
const now = fdu();

if (event.isAfter(now)) {
  console.log("Event is in the future!");
}
```
