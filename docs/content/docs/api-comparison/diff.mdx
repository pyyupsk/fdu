---
title: diff()
description: Calculate the difference between two dates
---

Calculates the difference between two dates in the specified unit. Returns a positive number if the current date is later than the compared date, negative if earlier.

## Syntax

```typescript
diff(other: FduInstance, unit?: UnitType): number
```

## Parameters

- **other**: `FduInstance` - Date instance to compare with
- **unit** (optional): `UnitType` - Unit of difference
  - `"year"` or `"y"` - Years
  - `"month"` or `"M"` - Months
  - `"week"` or `"w"` - Weeks
  - `"day"` or `"d"` - Days
  - `"hour"` or `"h"` - Hours
  - `"minute"` or `"m"` - Minutes
  - `"second"` or `"s"` - Seconds
  - `"millisecond"` or `"ms"` - Milliseconds (default)

## Returns

Returns the difference as an integer number. Positive if the current instance is later than the compared date, negative if earlier.

## Examples

### Difference in Days

```typescript
import { fdu } from "@pyyupsk/fdu";

const date1 = fdu("2025-09-30");
const date2 = fdu("2025-10-01");

date1.diff(date2, "day"); // -1
date2.diff(date1, "day"); // 1
```

### Difference in Months

```typescript
const date1 = fdu("2025-09-30");
const date2 = fdu("2025-12-30");

date1.diff(date2, "month"); // -3
date2.diff(date1, "month"); // 3
```

### Difference in Hours

```typescript
const date1 = fdu("2025-09-30T10:00:00");
const date2 = fdu("2025-09-30T14:30:00");

date1.diff(date2, "hour"); // -4
date2.diff(date1, "hour"); // 4
```

### Difference in Milliseconds

```typescript
const date1 = fdu("2025-09-30T10:00:00.000");
const date2 = fdu("2025-09-30T10:00:01.500");

date1.diff(date2); // -1500
date1.diff(date2, "millisecond"); // -1500
date1.diff(date2, "second"); // -1
```

### Practical Examples

```typescript
// Time until event
const now = fdu();
const event = fdu("2025-12-31");
const daysUntil = now.diff(event, "day");

// Age calculation
const birthday = fdu("2000-01-01");
const age = birthday.diff(fdu(), "year");

// Duration
const start = fdu("2025-09-30T09:00:00");
const end = fdu("2025-09-30T17:30:00");
const hours = end.diff(start, "hour"); // 8
```
