---
title: isSame()
description: Check if the date is the same as another date
---

Checks if the current date-time instance is the same as another date, either by exact timestamp or by a specified unit (year, month, day, etc.).

## Syntax

```typescript
isSame(other: FduInstance, unit?: UnitType): boolean
```

## Parameters

- **other**: `FduInstance` - Date instance to compare with
- **unit** (optional): `UnitType` - Unit of comparison
  - `"year"` or `"y"` - Same year
  - `"month"` or `"M"` - Same month and year
  - `"day"` or `"d"` - Same day, month, and year
  - `"hour"` or `"h"` - Same hour, day, month, and year
  - `"minute"` or `"m"` - Same minute, hour, day, month, and year
  - `"second"` or `"s"` - Same second (and all above)
  - If omitted, compares exact timestamps (millisecond precision)

## Returns

Returns `true` if the dates are the same, `false` otherwise.

## Examples

### Exact Comparison

```typescript
import { fdu } from "@pyyupsk/fdu";

const date1 = fdu("2025-09-30T14:30:00");
const date2 = fdu("2025-09-30T14:30:00");

date1.isSame(date2); // true
```

### Same Day

```typescript
const date1 = fdu("2025-09-30T10:00:00");
const date2 = fdu("2025-09-30T18:00:00");

date1.isSame(date2, "day"); // true
date1.isSame(date2); // false (different times)
```

### Same Month

```typescript
const date1 = fdu("2025-09-01");
const date2 = fdu("2025-09-30");

date1.isSame(date2, "month"); // true
date1.isSame(date2, "day"); // false
```

### Same Year

```typescript
const date1 = fdu("2025-01-01");
const date2 = fdu("2025-12-31");

date1.isSame(date2, "year"); // true
date1.isSame(date2, "month"); // false
```
