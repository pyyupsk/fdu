---
title: registerLocale()
description: Register a locale for use with fdu
---

Registers a locale configuration for date formatting. Locales must be registered before they can be used with `locale()` or `.locale()` methods.

## Syntax

```typescript
registerLocale(name: string, config: LocaleConfig): void
```

## Parameters

- **name**: `string` - Unique identifier for the locale (e.g., `"en"`, `"es"`, `"fr"`)
- **config**: `LocaleConfig` - Locale configuration object with:
  - `name`: `string` - Locale identifier
  - `months`: `readonly string[12]` - Full month names array
  - `monthsShort` (optional): `readonly string[12]` - Abbreviated month names
  - `weekdays`: `readonly string[7]` - Full weekday names (Sunday first)
  - `weekdaysShort` (optional): `readonly string[7]` - Abbreviated weekday names
  - `weekdaysMin` (optional): `readonly string[7]` - Minimal weekday names
  - `weekStart` (optional): `0-6` - First day of week (0=Sunday, 1=Monday, etc.)
  - `ordinal` (optional): `(n: number) => string` - Ordinal formatter
  - `meridiem` (optional): `(hour: number, minute: number, isLower?: boolean) => string` - AM/PM formatter

## Examples

### Register Built-in Locale

```typescript
import { registerLocale } from "@pyyupsk/fdu";
import { es } from "@pyyupsk/fdu/locale/es";

registerLocale("es", es);
```

### Register Multiple Locales

```typescript
import { registerLocale } from "@pyyupsk/fdu";
import { es } from "@pyyupsk/fdu/locale/es";
import { fr } from "@pyyupsk/fdu/locale/fr";
import { de } from "@pyyupsk/fdu/locale/de";

registerLocale("es", es);
registerLocale("fr", fr);
registerLocale("de", de);
```

### Override Existing Locale

```typescript
import { registerLocale } from "@pyyupsk/fdu";

// Register custom English variant
registerLocale("en-custom", {
  months: [
    "Januar",
    "Februar",
    "March", // ... custom names
  ],
  // ... other config
});
```

## Built-in Locales

fdu includes these built-in locales:

- `en` - English (registered by default)
- `es` - Spanish
- `fr` - French
- `de` - German
- `it` - Italian
- `pt` - Portuguese
- `ru` - Russian
- `ja` - Japanese
- `ko` - Korean
- `zh-cn` - Chinese (Simplified)
- `th` - Thai

Import and register as needed:

```typescript
import { registerLocale } from "@pyyupsk/fdu";
import { es } from "@pyyupsk/fdu/locale/es";
import { fr } from "@pyyupsk/fdu/locale/fr";

registerLocale("es", es);
registerLocale("fr", fr);
```

## Complete Example

```typescript
import { fdu, registerLocale, locale } from "@pyyupsk/fdu";
import { es } from "@pyyupsk/fdu/locale/es";
import { fr } from "@pyyupsk/fdu/locale/fr";

// Register locales
registerLocale("es", es);
registerLocale("fr", fr);

// Set global locale
locale("es");

// Use locales
const date = fdu("2025-01-15");

console.log(date.format("MMMM DD, YYYY"));
// 'enero 15, 2025'

console.log(date.locale("fr").format("DD MMMM YYYY"));
// '15 janvier 2025'
```
