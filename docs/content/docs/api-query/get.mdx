---
title: Get Components
description: Get individual date/time components
---

Retrieves individual date and time components from an fdu instance. All getter methods return numeric values representing the specific component.

## Methods

All getter methods return a number representing the component value. These methods correspond to the underlying JavaScript Date object getters.

### year()

Get the year.

```typescript
year(): number
```

**Example:**

```typescript
import { fdu } from "@pyyupsk/fdu";

const date = fdu("2025-09-30");
date.year(); // 2025
```

### month()

Get the month (0-11, where 0 = January).

```typescript
month(): number
```

**Example:**

```typescript
const date = fdu("2025-09-30");
date.month(); // 8 (September, 0-indexed)
```

### date()

Get the day of the month (1-31).

```typescript
date(): number
```

**Example:**

```typescript
const date = fdu("2025-09-30");
date.date(); // 30
```

### day()

Get the day of the week (0-6, where 0 = Sunday).

```typescript
day(): number
```

**Example:**

```typescript
const date = fdu("2025-09-30");
date.day(); // Returns 0-6 (depends on actual day of week)
```

### hour()

Get the hour (0-23).

```typescript
hour(): number
```

**Example:**

```typescript
const date = fdu("2025-09-30T14:30:45");
date.hour(); // 14
```

### minute()

Get the minute (0-59).

```typescript
minute(): number
```

**Example:**

```typescript
const date = fdu("2025-09-30T14:30:45");
date.minute(); // 30
```

### second()

Get the second (0-59).

```typescript
second(): number
```

**Example:**

```typescript
const date = fdu("2025-09-30T14:30:45");
date.second(); // 45
```

### millisecond()

Get the millisecond (0-999).

```typescript
millisecond(): number
```

**Example:**

```typescript
const date = fdu("2025-09-30T14:30:45.123");
date.millisecond(); // 123
```

## Complete Example

```typescript
import { fdu } from "@pyyupsk/fdu";

const date = fdu("2025-09-30T14:30:45.123");

console.log(date.year()); // 2025
console.log(date.month()); // 8 (September, 0-indexed)
console.log(date.date()); // 30
console.log(date.day()); // Day of week (0-6)
console.log(date.hour()); // 14
console.log(date.minute()); // 30
console.log(date.second()); // 45
console.log(date.millisecond()); // 123
```

## Practical Usage

```typescript
// Build custom date displays
const date = fdu("2025-09-30T14:30:00");
const customFormat = `Year: ${date.year()}, Month: ${date.month() + 1}, Day: ${date.date()}`;
// "Year: 2025, Month: 9, Day: 30"

// Conditional logic based on date components
if (date.hour() >= 12) {
  console.log("Afternoon");
}

// Extract specific components for calculations
const isWeekend = date.day() === 0 || date.day() === 6;
```
