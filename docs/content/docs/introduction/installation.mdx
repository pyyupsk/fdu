## Package Manager

Install fdu using your preferred package manager:

<Tabs items={["npm", "pnpm", "yarn", "bun"]}>
  <Tab value="npm">
  ```bash
  npm install @pyyupsk/fdu
  ```
  </Tab>
  <Tab value="pnpm">
  ```bash
  pnpm add @pyyupsk/fdu
  ```
  </Tab>
  <Tab value="yarn">
  ```bash
  yarn add @pyyupsk/fdu
  ```
  </Tab>
  <Tab value="bun">
  ```bash
  bun add @pyyupsk/fdu
  ```
  </Tab>
</Tabs>

## CDN

You can also use fdu via CDN for quick prototyping:

### ESM (Recommended)

```html
<script type="module">
  import { fdu } from 'https://esm.sh/@pyyupsk/fdu';

  const date = fdu();
  console.log(date.format('YYYY-MM-DD'));
</script>
```

### UMD (Browser Global)

```html
<script src="https://unpkg.com/@pyyupsk/fdu/dist/umd/index.iife.js"></script>
<script>
  const { fdu } = window.fdu;
  const date = fdu();
  console.log(date.format('YYYY-MM-DD'));
</script>
```

## Requirements

### Runtime

<Callout type="info">
**Note:** Specific runtime compatibility has not been thoroughly tested yet. The library is built with ESNext target and should work in modern environments with ESM support.
</Callout>

- **Node.js**: Modern version with ESM support recommended
- **Browsers**: Modern browsers with ESM support

### Development

To contribute or build from source, you'll need:

- **TypeScript**: 5.x
- **Build Tool**: [tsdown](https://tsdown.dev/)
- **Test Runner**: [Vitest](https://vitest.dev/)
- **Linter/Formatter**: [Biome](https://biomejs.dev/)
- **Package Manager**: [bun](https://bun.sh/)

## Verify Installation

After installation, verify fdu is working:

```typescript
import { fdu } from '@pyyupsk/fdu';

console.log(fdu().format('YYYY-MM-DD HH:mm:ss'));
// Output: "2025-10-05 15:30:00" (current date/time)
```
