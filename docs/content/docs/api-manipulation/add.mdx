---
title: add()
description: Add time to the date
---

Adds a specified amount of time to the date-time instance and returns a new instance. The original instance remains unchanged (immutable).

## Syntax

```typescript
add(value: number, unit: UnitType): FduInstance
```

## Parameters

- **value**: `number` - Amount to add (can be positive or negative)
- **unit**: `UnitType` - Unit of time
  - `"year"` or `"y"` - Years
  - `"month"` or `"M"` - Months
  - `"week"` or `"w"` - Weeks
  - `"day"` or `"d"` - Days
  - `"hour"` or `"h"` - Hours
  - `"minute"` or `"m"` - Minutes
  - `"second"` or `"s"` - Seconds
  - `"millisecond"` or `"ms"` - Milliseconds

## Returns

Returns a new `FduInstance` with the time added. The original instance is not modified.

## Examples

### Add Days

```typescript
import { fdu } from "@pyyupsk/fdu";

const date = fdu("2025-09-30");

date.add(1, "day").format("YYYY-MM-DD"); // '2025-10-01'
date.add(7, "d").format("YYYY-MM-DD"); // '2025-10-07'
```

### Add Months

```typescript
const date = fdu("2025-09-30");

date.add(1, "month").format("YYYY-MM-DD"); // '2025-10-30'
date.add(3, "M").format("YYYY-MM-DD"); // '2025-12-30'
```

### Add Years

```typescript
const date = fdu("2025-09-30");

date.add(1, "year").format("YYYY-MM-DD"); // '2026-09-30'
date.add(5, "y").format("YYYY-MM-DD"); // '2030-09-30'
```

### Add Time

```typescript
const date = fdu("2025-09-30T14:30:00");

date.add(1, "hour").format("HH:mm"); // '15:30'
date.add(30, "minute").format("HH:mm"); // '15:00'
date.add(45, "second").format("HH:mm:ss"); // '14:30:45'
```

### Chaining

```typescript
const date = fdu("2025-09-30");

date.add(1, "month").add(5, "day").add(2, "hour").format("YYYY-MM-DD HH:mm"); // '2025-11-04 02:00'
```
