---
title: diff()
description: Calculate the difference between two dates
---

Calculate the difference between two dates.

## Syntax

```typescript
diff(date: ConfigType, unit?: OpUnitType, float?: boolean): number
```

## Parameters

- **date**: `ConfigType` - Date to compare with (`string | Date | number | Fdu`)
- **unit** (optional): `OpUnitType` - Unit of difference
  - `"year"` or `"y"`
  - `"month"` or `"M"`
  - `"day"` or `"d"`
  - `"hour"` or `"h"`
  - `"minute"` or `"m"`
  - `"second"` or `"s"`
  - `"millisecond"` or `"ms"`
  - Default: `"millisecond"`
- **float** (optional): `boolean` - Return decimal value (default: `false`)

## Returns

Returns the difference as a number. Positive if the date is later, negative if earlier.

## Examples

### Difference in Days

```typescript
import { fdu } from "@pyyupsk/fdu";

const date1 = fdu("2025-09-30");
const date2 = fdu("2025-10-01");

date1.diff(date2, "day"); // -1
date2.diff(date1, "day"); // 1
```

### Difference in Months

```typescript
const date1 = fdu("2025-09-30");
const date2 = fdu("2025-12-30");

date1.diff(date2, "month"); // -3
date2.diff(date1, "month"); // 3
```

### Difference in Hours

```typescript
const date1 = fdu("2025-09-30T10:00:00");
const date2 = fdu("2025-09-30T14:30:00");

date1.diff(date2, "hour"); // -4
date1.diff(date2, "hour", true); // -4.5 (with decimals)
```

### Difference in Milliseconds

```typescript
const date1 = fdu("2025-09-30T10:00:00.000");
const date2 = fdu("2025-09-30T10:00:01.500");

date1.diff(date2); // -1500
date1.diff(date2, "millisecond"); // -1500
date1.diff(date2, "second"); // -1
date1.diff(date2, "second", true); // -1.5
```

### Practical Examples

```typescript
// Time until event
const now = fdu();
const event = fdu("2025-12-31");
const daysUntil = now.diff(event, "day");

// Age calculation
const birthday = fdu("2000-01-01");
const age = birthday.diff(fdu(), "year");

// Duration
const start = fdu("2025-09-30T09:00:00");
const end = fdu("2025-09-30T17:30:00");
const hours = start.diff(end, "hour", true); // 8.5
```
