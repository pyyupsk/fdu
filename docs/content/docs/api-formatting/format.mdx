Format date to string with custom pattern.

## Syntax

```typescript
format(pattern?: string): string
```

## Parameters

- **pattern** (optional): `string`
  - Format pattern using tokens
  - Default: `"YYYY-MM-DDTHH:mm:ssZ"` (ISO 8601)

## Returns

Returns formatted date string.

## Examples

### Basic Formatting

```typescript
import { fdu } from "@pyyupsk/fdu";

const date = fdu("2025-09-30T14:30:45");

date.format(); // '2025-09-30T14:30:45Z'
date.format("YYYY-MM-DD"); // '2025-09-30'
date.format("DD/MM/YYYY"); // '30/09/2025'
```

### Date Formats

```typescript
date.format("YYYY-MM-DD"); // '2025-09-30'
date.format("YY-MM-DD"); // '25-09-30'
date.format("MMMM DD, YYYY"); // 'September 30, 2025'
date.format("MMM DD, YY"); // 'Sep 30, 25'
date.format("DD MMMM YYYY"); // '30 September 2025'
```

### Time Formats

```typescript
date.format("HH:mm:ss"); // '14:30:45'
date.format("hh:mm:ss A"); // '02:30:45 PM'
date.format("HH:mm"); // '14:30'
date.format("h:mm a"); // '2:30 pm'
```

### Combined Formats

```typescript
date.format("YYYY-MM-DD HH:mm:ss"); // '2025-09-30 14:30:45'
date.format("MMM DD, YYYY [at] hh:mm A"); // 'Sep 30, 2025 at 02:30 PM'
date.format("dddd, MMMM DD, YYYY"); // 'Monday, September 30, 2025'
```

### Escaped Text

```typescript
date.format("[Today is] YYYY-MM-DD"); // 'Today is 2025-09-30'
date.format("YYYY [year]"); // '2025 year'
date.format("[The time is] HH:mm"); // 'The time is 14:30'
```
